# Google Gemini APIキーの取得方法

このアプリでは、手書き文字の認識と採点にGoogleのGemini APIを使用しています。
以下の手順に従ってAPIキーを取得し、アプリに設定してください。

## 1. Google AI Studioにアクセス
ブラウザで以下のURLにアクセスしてください。
[https://aistudio.google.com/](https://aistudio.google.com/)

## 2. Googleアカウントでログイン
画面右上の「Sign in」ボタンなどから、お使いのGoogleアカウントでログインしてください。

## 3. APIキーの作成
1. 画面左上の「Get API key」ボタンをクリックします。
2. 「Create API key」ボタンをクリックします。
3. 「Create API key in new project」（新しいプロジェクトで作成）を選択します。
   ※既存のGoogle Cloudプロジェクトがある場合は、そちらを選択しても構いません。

## 4. キーのコピー
APIキーが生成されると、画面に表示されます。「Copy」ボタンを押してキーをコピーしてください。
※このキーは他人に教えないようにしてください。

## 5. アプリへの設定
1. このアプリのトップページ（`index.html`）を開きます。
2. 画面右上の歯車アイコン（設定）をクリックします。
3. 「Gemini APIキー」の入力欄に、先ほどコピーしたキーを貼り付けます。
4. 「保存」ボタンをクリックします。

以上で設定は完了です。テスト画面で「AIカガワ先生に聞く」ボタンが使えるようになります。

## 注意事項
- Gemini APIには無料枠がありますが、大量に使用すると制限がかかる場合があります。
- APIキーの取り扱いには十分ご注意ください。
